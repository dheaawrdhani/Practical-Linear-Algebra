# Practical Linear Algebra for Data Science - Book Reproduction

## Overview
Repository ini berisi reproduksi dan rangkuman materi dari buku *Practical Linear Algebra for Data Science* karya Mike X. Cohen.

## Objectives
- Memahami konsep dasar Linear Algebra untuk Data Science.
- Mengimplementasikan contoh-contoh menggunakan Python.
- Mempelajari penerapan Linear Algebra pada Machine Learning dan Data Analysis.

## Topics Covered
- Vectors and Vector Operations
- Matrices and Matrix Operations
- Matrix Inverse
- Linear Systems of Equations
- Orthogonality
- LU Decomposition
- QR Decomposition
- Eigenvalues and Eigenvectors
- Singular Value Decomposition (SVD)
- Principal Component Analysis (PCA)

## Tools
- Python
- NumPy
- SciPy
- Google Colab / Jupyter Notebook

## Key Insights
Linear Algebra merupakan dasar penting dalam Data Science karena digunakan dalam:
- Data representation
- Dimensionality reduction
- Machine Learning algorithms
- Feature extraction
- Data visualization
